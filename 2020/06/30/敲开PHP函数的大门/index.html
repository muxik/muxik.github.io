函数结构

构成部分:

  • 关键字 function
  • 函数名 functionName
  • 参数列表 $a,$b
  • 函数体 {}
  • 返回值 return $a,$b

例：

function functionName($a,$b)
{
    return $a+$b;
}

函数参数

主要有以下几类：

  • 形参&实参
  • 默认值
  • 强类型参数
  • 可变数量的参数列表
  • 值传递&引用传递

形参&实参

形参:

function sum($x,$y /*形参*/){
    $tmp = $x + $y
    return $tmp
}

实参

function sum($x,$y){
    $tmp = $x + $y
    return $tmp
}
$x = 1;
$y = 2;
sum($x,$y /*实参*/)

注意:

调用函数时，形参和实参的关系是一对一的,实参按顺序赋值给形参，实参可以是一个表达式，传递实参大于形参数量不会报错，反之小于形参会报错。

默认值

解释:允许函数的参数在调用的时候给定具体的值，如果没有对应的值函数可以是使用默认值作为变量，反之如果有则会替换默认值,缺省值应该放在右边，默认值必须是常量

可变数量的参数列表

code/args_list.php

实现方式 1:

在 php5.5 及更早的版本中，使用 func_num_args(), func_get_arg(), func_get_args()函数实现。

  • func_num_args()
    • 返回用户传递参数的数量
  • func_get_arg()
    • 返回参数列表中的某一项(是一个偏移量从 0 开始如果指定偏移量大于传递偏移量会有警告)
  • func_get_args()
    • 返回包含函数参数列表的数组
function get_sum()
{
    $args_num = func_num_args();
    $sum = 0;
    if ($args_num == 0) {
        return 0;
    } else {
        for ($i = 0; $i < $args_num; $i++) {
            $sum += func_get_arg($i);
        }
        return $sum;
    }
}

echo get_sum(1,2,4);

实现方式 2:

在 php5.6 及以上的版本中，可以使用…语法实现,返回一个数组。

function get_sum(...$args)
{
    if ($args == []) {
        return 0;
    } else {
        for ($i = 0; $i < count($args); $i++) {
            $sum += $args[$i];
        }
        return $sum;
    }
}

echo get_sum(1, 2, 3);

值传递&引用传递

code/args_pass.php

1. 值传递:

function factorial($num)
{
    $num = $num * 2;
    return $num;
}

$num = 3;
echo factorial($num);
echo $num;
// 此时我们发现函数并不会影响实际的变量$num
}

1. 引用传递:


function swap(&$x, &$y)
{
    $tmp = $x;
    $x = $y;
    $y = $tmp;
}


$a = 1;$b = 2;
swap($a, $b);

echo "\n" . $a, $b;
// 此时我们发现变量$a,$b值发生了交换

代码复用

如果你想在一个文件里面引入其他文件的函数
可以使用一下几种方法

  • include()
  • require()

include 和 require 语句是相同的，除了错误处理方面：

require 会生成致命错误（E_COMPILE_ERROR）并停止脚本
include 只生成警告（E_WARNING），并且脚本会继续

  • include_once()
  • require_once()

表示文件只能引入一次
